# 📋 问卷助手 - Chrome 浏览器插件

高效填写问卷调查的 Chrome 扩展，提供键盘快捷键、快速填片、陷阱题提醒和进度追踪功能。

## 功能

| 功能 | 说明 |
|------|------|
| ⌨️ **键盘快捷键** | 数字键选选项，Tab 翻页，全程无需鼠标 |
| 📝 **快速填片** | 预设姓名/年龄/城市等常用信息，一键填入 |
| ⚠️ **陷阱题提醒** | 自动检测注意力检测题，红色高亮警告 |
| 📊 **进度追踪** | 顶部进度条 + 预计剩余时间 |
| 🔄 **矛盾检测** | 前后回答矛盾时提示（如"不吸烟"→"每天几包"） |

## 安装方法

1. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/`
2. 打开右上角「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择本项目根目录 `wenjuan/`
5. 插件安装完成！

## 使用方法

### 快捷键速查

| 快捷键 | 功能 |
|--------|------|
| `1` - `9` | 选择对应选项 |
| `1` `2` | 连续按选第 12 项 |
| `0` | 清除当前题选择 |
| `Tab` / `Enter` | 下一题/下一页 |
| `Shift` + `Tab` | 上一题 |
| `Ctrl` + `Shift` + `F` | 弹出快速填片面板 |
| `↑` `↓` / `j` `k` | 上下切换题目 |

### 管理快速填片

1. 点击插件图标 → 「⚙️ 设置」
2. 在「快速填片」标签页中添加/编辑常用信息
3. 填问卷时遇到输入框，按 `Ctrl+Shift+F` 快速选择

### 功能开关

点击插件图标，可随时开关各功能模块。

## 隐私说明

- 所有数据存储在浏览器本地（chrome.storage.sync）
- **不会**上传任何问卷内容或个人信息
- **不会**追踪你的浏览行为
- 仅在你访问问卷页面时激活

## 技术栈

- Chrome Extension Manifest V3
- 原生 JavaScript（无依赖）
- MutationObserver（监听动态 DOM）
- chrome.storage.sync（跨设备同步设置）
